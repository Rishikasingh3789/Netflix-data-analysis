# Netflix Data Analysis 📊

This project analyzes Netflix dataset to uncover trends and insights.

## 📊 Key Insights:
- Movies dominate over TV Shows  
- USA produces the highest content  
- Content growth increased rapidly after 2015  
- Drama and Comedy are popular genres  

## 🛠️ Tools Used:
- Python  
- Pandas  
- Matplotlib  

## 📁 Project Files:
- netflix-data-analysis.ipynb  
- netflix_titles.csv
